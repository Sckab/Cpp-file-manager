# C++ File Manager

</div>

This is a simple file manager made in C++, this is just for learning, i hope you like it!

| Command          | Purpose                        |
| ---------------- | ------------------------------ |
| `make main`      | For just compiling it          |
| `make build-run` | For compiling and running it   |
| `make clean`     | For cleaning the bin directory |
